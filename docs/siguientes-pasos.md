# Siguientes Pasos (Post-MVP)

1. **Grabación de Sesiones:** Integrar la funcionalidad de grabación nativa de Daily.co.
2. **Múltiples Salas:** Permitir que los usuarios creen sus propias salas con IDs únicos.
3. **Moderación:** Implementar roles de moderador que puedan silenciar o expulsar usuarios via RTDB.
4. **Calidad Adaptativa:** Ajustar la configuración de Daily para optimizar el ancho de banda según el número de participantes.
5. **Estadísticas Avanzadas:** Graficar la retención de audiencia y picos de conexión.
